    Data sourcing

    Real data

    Segment loss-ratio benchmarks (plan LRs)
    Sourced from NAIC 2023 P&C Annual Report and NCCI 2024 State of the Line Guide. Transportation reflects the commercial auto crisis (CY combined ratio 113%, NAIC 2023). Workers comp-exposed lines reflect NCCI's 86% combined ratio / ~55% loss ratio.

    Real data

    Claim frequency & severity parameters
    Industry-sector frequency and average claim severity based on ISO/Verisk actuarial analyses and NCCI class-code experience data. Construction: high-frequency / moderate-severity. Transportation: lower-frequency / high-severity commercial auto.

    Simulated

    Individual policies, brokers & monthly series
    Policy-level data is generated by a seeded Monte Carlo simulation using the real benchmarks above as parameters. Status flags use z ≥ 2.5 or |var| ≥ 5pp. AI narration via Claude API (Anthropic).

    Plan loss ratios sourced from NAIC 2023 P&C Annual Report · NCCI 2024 State of the Line · Verisk GL Actuarial Hub. What-if narration powered by Claude (Anthropic). No PII; no live system access.
